# Socket
configure a TCP/IP socket in C language to play minefield at local network.


> To compile

```make```

> To run a local IPv4 server

```make server_v4```

> To run a local IPv6 server

```make server_v6```

> To run a local IPv4 client

```make client_v4```

> To run a local IPv6 client

```make client_v6```
